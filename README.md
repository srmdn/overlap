# Overlap

See what time it is for your global team — right now.

Overlap shows live clocks for your key time zones and tells you whether each teammate is likely working, in early hours, or asleep. Includes a time converter for scheduling meetings across zones.

Built for people who work with remote teams and are tired of timezone confusion.

## Features

- Live clocks for New York, London, Jakarta, and Sydney
- "Awake?" status per zone — working, early hours, after work, or sleeping
- Time converter — pick any time and source zone, see all conversions instantly

## Stack

React · TypeScript · Vite · Tailwind CSS

## Development

```bash
git clone https://github.com/srmdn/overlap.git
cd overlap
npm install
npm run dev
```
